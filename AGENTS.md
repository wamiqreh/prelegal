# Project: Local Event & Workshop Booking Platform

## Business Concept
- Small businesses, community centers, and freelancers host workshops, classes, or events (e.g., cooking classes, coding bootcamps, fitness sessions).  
- Platform allows users to **discover local events**, **book spots online**, and **pay or reserve seats**.  
- Organizers can **create and manage events**, **view attendee lists**, and **send updates**.  
- Revenue model: **commission per booking** or **subscription for premium features**.

## Core Features

### User Features
- Browse events by category, date, or location
- Book and cancel reservations
- View profile and booking history

### Organizer Features
- Create, edit, and delete events
- Track attendees
- Mark events as free or paid

### Admin Features
- Approve new events
- Manage users and organizers
- View platform statistics
